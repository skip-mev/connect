# Kucoin Provider

## Overview

The Kucoin provider is utilized to fetch pricing data from the Kucoin web socket API. You need to apply for one of the two tokens below to create a websocket connection. It should be noted that: if you subscribe to spot/margin data, you need to obtain tokens through the spot base URL; if you subscribe to futures data, you need to obtain tokens through the futures base URL, which cannot be mixed.
